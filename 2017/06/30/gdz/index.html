1. 表面的获得

  • 表面的研磨

  • 表面的抛光

    • 化学抛光

      • 定义：酸或碱溶液选择性地溶解表面凸出部分

      • 酸或碱溶液+缓冲剂

        • 目的：可控

        • 分类

          • 化学缓冲：加入弱酸（弱碱）或弱酸（弱碱）性物质，控制抛光液的 PH 值
          • 物理缓冲：增大抛光液的粘度，减弱其流动性加入粘稠性 物质实现，如甘油等。

      • 原理：凸处的扩散层较薄，溶液的浓度梯度越大，扩散速度大，溶解速度加快

      • 工艺特点：效率较高，但效果较差。

      • 例子

        • Si 或 Ge

          • 抛光液： HNO3 或 HF
          • 缓冲剂：冰醋酸，加入少许 Br

        • GaAs 系

          • 抛光液： Cl2 或 Br2 溶入甲醇
          • 缓冲剂：冰醋酸

    • 机械抛光

      • 定义：选用细分的金属氧化物利用机械摩擦过程，清除凹凸不平的凸出部分

      • 问题

        • 需要水冷却
        • 易使表面发生塑形形变

      • 工艺特点：效果较好，但效率较低，易损伤表面

    • 化学 - 机械抛光

      • 定义：化学抛光与机械抛光的有机结合

      • 抛光剂：碱性 SiO2 胶体溶液

      • 原理：Si 晶片表面首先与抛光液发生化学反应

      • 例子：半导体 Si 晶片的化-机械抛光工艺

        • 化学抛光

          • Si晶片表面首先与抛光液发生化学反应
          • Si+2NaOH+H2O → Na2SiO3+2H2↑
          • Si+2NaOH+2H2O→Na2SiO4+3H2↑

        • 机械抛光

          • 使用碱性 SiO2 胶体溶液作为抛光剂，主要针对 Na2SiO3 进一步提高反应速率，同时对 Si 进行软摩擦

      • 原则：化学反应与机械摩擦循环进行，共同作用，达到抛光的目的

      • 工艺特点：抛光效果较好，效率亦较高

2. 表面的清洁处理

  • 湿法清洁

    • 表面去油

      • 有机溶剂去油

      • 碱液去油

      • 合成洗涤剂去油

      • 电解去油

        • 电解液：NaOH 溶液
        • 目的：金属表面去油

      • 超声清洗

        • 原理：超声波的空化作用

          • 使清洗液介质产生机械振动
          • 介质质点产生相互交替的稠密和稀疏的变化
          • 稠密处液体受到压力，稀疏处液体受到拉力
          • 从而产生强大的冲击力，使表面的油污等污染物冲掉，达到清洗的目的。

        • 超声清洗液

          • 条件

            • 具有能溶解油污的能力
            • 容易产生气泡

          • 例子：汽油、丙酮、乙醇

    • 表面氧化层的清除

      • 定义：用酸或碱的溶液去除表面氧化层或其它腐蚀产物

      • 例子

        • Fe 表面的氧化铁可用盐酸去除
        • Si 表面的 SiO2 层可用氢氟酸去除

  • 干法清洁

    • 定义：一般表示在真空环境下利用物理或物理 / 化学 作用的清洁处理方法

    • 目的

      • 清除表面吸附物
      • 提高薄膜与衬底材料表面之间的附着力
      • 改善表面的应力分布

    • 分类

      • 烧氢与真空热处理

        • 烧氢

          • 定义：金属氧化物在氢气环境下的还原过程
          • 化学方程式：MeO+H2→Me+H2O

        • 真空热处理

          • 定义：金属氧化物在高温及真空条件下的分解过程
          • 化学方程式：MeO→Me+O2

        • 注意

          • 高温条件
          • 一般不能用于功能材料

      • 加热与辐射处理

        • 表面加热处理

          • 定义：在真空条件下加热衬底表面，会促使吸附的水分子和各种碳氢化合物分子的解吸作用有不同程度的增强
          • 加热温度：100-800℃
          • 加热时间：10-60分钟

        • 表面辐射处理

          • 利用紫外辐射分解表面吸附的碳氢化合物

          • 原理：表面污染物中的油脂类或其他有机溶剂所含有的碳氢化合物，吸收高能量的紫外光能量后，使其激发并分解成气相的小分子，如 CO2，H2O ，N2 等

          • 改进

            • 臭氧环境下，清洁效果更好，效率更高
            • 原因：因为臭氧的存在会产生高活性的原子态氧，受激的污染物分子和由污物离解产生的自由基与原子态氧作用，形成较简单的易挥发分子

      • 电子束与离子束轰击清洗

        • 电子束轰击清洗

          • 工艺过程

            • 电子束轰击表面时，还会发生电子诱导脱附效应，以及表面某些吸附物的分解，从而起到清洁表面的作用。
            • 具有一定能量的电子束轰击表面时，其动能绝大部分转变成热能，使表面温度上升，从而促使挥发性污染物脱附并蒸发掉

          • 工艺特点

            • 加热仅在表面发生，所以升温快，清洗时间短， 零部件的平均热负荷低
            • 电子束能量不宜过高，否则易造成表面辐射损伤
            • 在表面有碳存在时，电子轰击会在表面形成牢 固的污染物

        • 离子束轰击清洁

          • 工艺过程

            • 利用辉光放电产生的惰性气体正离子或其等离子体轰击待清洁表面
            • 这与上述的电子束轰击清洗类似，也会造成轰击处局部升温，使得表面污染物挥发掉

          • 分类

            • 辉光放电清洗（等离子体清洗）
            • 偏压溅射清洗（正离子溅射清洗）

  • 薄膜的生长与形成过程

    • 薄膜的生长

      • 气原子的吸附：失去垂直于表面的速度分量或能量 → 表面

      • 原子团（簇）的形成：热力学非平衡状态 → 解吸而重新蒸发

      • 凝结核的生长：与其它吸附原子或原子团碰撞结合 → 凝结核

      • 结晶核的产生与生长：克服了该凝结核的势垒 → 热力学稳定 → 凝结核 → 不断增大数量增加 → 饱和状态

        • 结晶核的取向影响到后续薄膜的晶体结构

    • 薄膜的形成

      • “小岛” 的形成：水平方向上的生长速度远大于垂直方向上的生长速度 → 小岛
      • “小岛”的聚结：为了降低表面能 → 较大尺寸的岛
      • “ 网状结构（网膜） ” 的形成：变为多孔性的网状结构
      • 连续薄膜的形成：将所出现的空洞和通道填满
      • 总结：结晶核越多 → 生长成连续薄膜的时间也就越短

        • 实际附着力：通过实验测量所获得的附着力

      • 定义：衡量薄膜与基片之间的相互作用强度（结合力 / 粘附力）

        • 增大入射蒸发原子或分子的吸附能量
        • 增大初始凝聚核的密度

        • 加热基片表面

      • 定义：用聚焦的高能量电子束轰击加热膜料的一部分，而使其气化蒸发的方法

      • 热电子发射的电流密度与金属温度的关系

      • 分类

        • 传统直线型电子枪

          • 缺点：材料利用率低

        • e型电子枪

          • 组成

          • 特点

            • 偏转磁场能有效抑制二次电子到达沉积表面，避免其对成膜过程的影响
            • 对导电性能较差的膜料，有效消除蒸发表面的电荷屏蔽效应
            • 阴极受到屏蔽，不受污染，工作稳定且寿命长
            • 聚焦电子束可在膜料表面以一定频率进行 X-Y 两维方向扫描，使得在较大面积蒸发面上加热更均匀

        • 工艺特点

          • 通过聚焦的高能量电子束的加热手段，能量高度集中，能使膜料局部表面获得极高的温度，因此可以蒸镀高熔点的材料
          • 可以准确且方便地控制蒸发温度，并有较大的温度调节范围，膜料的适用范围广泛
          • 加热仅发生在被镀材料表面，不需对坩埚 （可通冷却水）进行加热，避免了坩埚材料对膜料的反应与污染
          • 由于蒸发时能量密度大，所产生的蒸汽分子动能较大，可获得更致密、附着力更好的膜层

    • 离子束辅助蒸发镀膜

      • 定义：用宽束低能离子束轰击蒸发 沉积薄膜（衬底）的方法

      • 常规模式：采用惰性气体离子束辅助轰击

      • 低能原因：避免破坏

      • 解决的问题：为了改善蒸发沉积膜层结构与性能的不足

        • 生长出柱状或锥状晶粒薄膜结构
        • 表现出不同的薄膜内应力
        • 较低的机械强度与膜密度
        • 另外薄膜与衬底间的附着力也较低

      • 物理过程

        • 离子轰击从衬底表面脱附或溅出杂质，或脱附吸附力较弱的部分成膜原子或分子
        • 离子轰击增强沉积原子沿衬底表面的迁移扩散能力，影响成核率及薄膜生长，同时可增强沉积过程中的相关化学反应能力
        • 低能离子轰击也可影响生长薄膜的晶粒尺寸、优选晶面及取向度、提高薄膜晶体中的填充原子密度
        • 增强薄膜原子表面或体扩散能力、改变薄膜应力性质及大小、改变薄膜与衬底或薄膜与薄膜之间的界面层性质

    • 激光束加热蒸发

      • 定义：高功率的激光束作用于被镀材料表面，材料吸收光子能量后被加热气化并蒸发，然后凝结沉积在基片表面上形成薄膜

        • 定义：利用强功率的脉冲激光输出，使膜料瞬间蒸发并沉积膜层的

        • 过程

          • 激光与靶材的相互作用

            • 激光功率密度很高 → 物体表面蒸发
            • 相当多的原子被激发和离化 → 靶的表面形成高密度、高温度的致密等离子层
            • 最后形成不同的层

          • 烧蚀物在衬底表面的成膜

            • 等离子体的前向传输的特点：方向性与各项异性
            • 传输过程中能量快速衰减
            • 引入某种气体（O2，量少效果好） → 碰撞、散射、激发 → 影响烧蚀物到达衬底表面时的状态、数量、分布

            • 与薄膜的生长类似

          • 优点

            • 易于实现同成分沉积，即薄膜成分与靶材成分基本保持一致
              在制备复杂的多组分化合物薄膜方面具有 明显的优势
            • 高能量等离子体沉积，能量高达50eV 。 有助于良好结晶薄膜生长,和在气相中或 衬底表面完成化学反应
            • 易于在气氛中实现反应沉积过程

          • 缺点

            • 难以沉积大面积均匀薄膜 → 效率低

    • 其它蒸发镀膜技术

      • 多源蒸发法

        • 问题：对于合金或多组元成分的蒸发和沉积，会产生所谓的“分馏现象”。即沉积薄膜的化学配比会偏离原材料的化学配比。
        • 定义：将每种单质材料分别放在各自的蒸发源内，采取一定的加热方法同时加热蒸发，分别控制各蒸发源的蒸发温度与蒸发速率，使得各成分具有设计的沉积速率，最后获得适当的所需成分比例的多元化合物膜层。

      • 反应蒸发法

        • 问题：蒸发材料中含有易挥发成分，如氧化物中的氧，由于加热蒸发过程中，该类化合物往往产生部分电离或离解，造成在沉积成膜时易挥发成分 的缺失，以致膜层的化学成分偏离原材料
        • 定义：将某种活性气体导入真空室内，使该活性气体的原子或分子和从蒸发源逸出的的原子或分子在基片表面发生化学反应，制备所需化学配比的化合物薄膜
        • 例子：制备氧化物薄膜时，蒸发沉积过程中向真空室内充入微量的氧气

3. 溅射镀膜

    • 前提：在一定的真空条件下

    • 定义：利用高能量的惰性气体正离子撞击由待镀材料做成的靶材表面，通过离子与固体表面原子的动量交换，使表面原子克服其结合能而从表面逸出，溅射出来的原子通过迁移到达基片表面，最后沉积于基片表面上，从而形成薄膜。

      • 反常辉光放电：随着放电电流的增大，放电的电流密度与阴极电位降随之增大，这时撞击阴极的正离子数目及动能都比正常辉光放电时大为增加，在阴极表面发生的溅射作用也要强得多。

        • 离子能量转化为热能 → 需对靶进行冷却

    • 靶表面层的变化和扩散

      • 成分变化

        • “ 蚀变层 ”：溅射多组分靶材时，在同一溅射功率下，由于不同组分的溅射产额不相同，开始溅射时，溅射产额大的组分会发生优先溅射。这样即会使得靶表面层的成分变化。

        • “ 稳恒蚀变层 ” ：随着溅射的进行，优先溅射的成分浓度（或所占比例）降低，溅射产额小的成分浓度增加，有可能达到 “ 稳恒蚀变层 ”（即平衡状态）。

        • “ 预溅射 ”的作用

          • 解决膜层成分偏离靶材成分的问题：随着预溅射的进行，优先溅射的成 分浓度（或所占比例）降低，溅射产额小的成分浓度增加，有可能达到 “ 稳恒蚀变层 ”（即平衡状态） → 被溅射的成分对应于靶材的成分
          • 清除靶表面吸附气体，及其他表面污染物

      • 扩散过程

        • 类型

          • 入射离子的溅射引起的靶原子扩散
          • 由热量引起的靶原子表面扩散（迁移扩散）
          • 溅射气体的离子注入引起的靶原子扩散

        • 因素：靶材料相成分、晶粒尺寸、靶材性能

        • 影响

          • 平衡态的建立时间
          • 稳恒蚀变层的厚度

          • 靶材料（背面水冷）作为阴极，衬底（背部加热）作为阳极，两极之间加上直流电压

            • 阴极（靶材）：负交压
            • 阳极：接地

        • 工艺过程

        • 主要工艺参数的选择

        • 工艺特点

          • 优点：设备结构简单，可获得大面积均匀薄膜

          • 缺点

            • 溅射电压高

            • 沉积速率低

            • 基片升温高

            • 工作气压高

      • 射频溅射

        • 背景：直流二极溅射只能用于沉积金属膜，而不能沉积制备介质薄膜

        • 条件：射频（交流 / 高频）辉光放电

        • 区别

          • 电子在负辉区不断振动进行更多的碰撞电离 → 放电不太依赖于二次电子的发射 → 可降低工作气体压强
          • 高频电压能成为电容性耦合 → 可以以此溅射沉积介质膜

        • 基本原理

          • 概述：利用高频电磁辐射来维持低气压下的射频辉光放电

          • 在射频电压的一个周期内，正离子和电子可以交替地轰击靶面，从而实现溅射介质靶材的目的

            • 负半周期：正电荷的积累
            • 正半周期：中和了积累在介质靶表面上的正电荷

          • 常采用两个金属电极面积大小不等，即构成 “ 非对称平板型结构 ”
            根据
            阳极面积大到足以使流向它的离子能量小于溅射阈能，则在阳极（衬底）表面就不会发生溅射，仅起到离子轰击清洁的作用

            • 阴极：小电极，射频电源
            • 阳极：大电极，接地
            • 电压与离子能量作用关系

          • 基本条件

            • 靶材必须与射频电源电压之间发生电容性耦合（阻抗匹配）
            • 阴极（靶表面）面积必须小于阳极的表面积

        • 选择射频（高频）作为溅射的电源的原因

          • 在高频（ 10 ～ 30MHz ）电场作用下，等离子体中的电子容易在交流电场中振荡并吸收能量，因此，电子与工作气体分子碰撞并使之电离产生离 子的概率变大，故使得击穿电压、放电电压及工作气压显著降低。
          • 离子比电子质量大，迁移率小，所以靶表面的电位上升缓慢。由于在靶上会形成负偏压，所以射频溅射装置也可以溅射导体靶

        • 工艺特点

          • 可以用来溅射沉积包括介电材料在内的任何固体靶材料
          • 由于电子有效利用率的提高，可以获得较低的工作气体压强
          • 溅射沉积膜层的纯度高、致密性好、与基片的附着性能好
          • 具有较大的溅射率与沉积速率

        • 缺陷

          • 百分之几的气体被离化
          • 工作气体压强都较高 → 膜层纯度较低，沉积速率也较低
          • 二次电子的不断碰撞 → 基片升温

        • 背景：射频溅射的缺陷

        • 基本原理：在DC或RF溅射的基础上，通过外加一个垂直于靶表面（亦即垂直于电场方向）的磁场，此时处于正交电场和磁场中的电子的运动方程为：

          电子在洛伦兹力的作用下，以轮摆线的形式沿着靶表面运动

          • 后果

            • 增加了电子与惰性气体分子的碰撞几率 → 提高了电离效率 → 辉光放电等离子体密度增大 → 显著提高溅射率（沉积速率）

            • 高密度等离子体被磁场约束在靶表面附近，提高了沉积速率。

                工作气压低，减小了溅射原子或分子与气体分子的碰撞

        • 工艺特点

          • 低压：工作气压低
          • 低温：基片升温低
          • 高速：沉积速率高
          • 高纯：膜层纯度高

          • 无法制备单晶膜层
          • 溅射原子能量不同 → 结构复杂

          • 溅射：动量传递的离子轰击过程

            • 原子动能较高
            • 主要在较高轰击能量下按余弦定律分布
            • 对于合金或多元材料：优先溅射

        • 迁移过程

          • 蒸发

            • 蒸发粒子在（超）高真空迁移输运，不碰撞或很少碰撞

          • 溅射

            • 溅射粒子在工作气压相对中较高下迁移输运，大量碰撞减少能量
            • 电荷转移过程
            • 方向有明显变化（各项同性），很容易发生化学反应。

        • 凝结成膜过程

          • 蒸发

            • 入射原子对基片表面基本无影响
            • 成核条件不变
            • 残余气体原子或分子的弱入射
            • 膜层中气体掺入少（纯膜）与外来气体很少发生化学反应，基片与膜层的温度没有显著变化

          • 溅射

            • 入射离子和高能量中性粒子对基片表面影响强烈
            • 成核条件明显改变
            • 工作气体与残余气体原子、分子和离子的强入射
            • 外来气体掺杂较高，易发生化学反应，衬底与膜层有较大的温度变化没有显著变化

      • 特点

        • 蒸发镀膜

          • 膜层纯度较高，结晶性能较好
          • 致密性、机械强度、附着力较低

        • 溅射镀膜

          • 膜层致密性、机械强度、附着力较高
          • 膜层纯度较低，结构缺陷较高

      • 低压化学气相沉积

        • 背景：解决常压CVD存在的问题

        • 系统原理图

        • 重大进步

          • 结晶性能与纯度

            • 一定的真空条件 , 反应室压力减小
              → 反应气体分子的平均自由程增加，扩散常数增大
              → 减少不易挥发的还原或分解产物在气相中的凝结
              → 提高沉积层的结晶性能与纯度

          • 沉积速率

            • 气相分子扩散率的大幅增大
              → 反应物向衬底的气相迁移速率提高

              → 提高了反应气体的有效利用率和沉积速率

              → 通过改变扩散方式与进程
              → 常压下的 “ 流线型沉积 ” 改善为较大面积的均匀沉积
              → 提高了沉积膜层的均匀性

      • 等离子体增强化学气相沉积（PECVD）

        • 基本原理：利用射频辉光放电所产⽣的等离⼦体的活性，使得反应⽓体受激、离解或分解，提⾼反应活性，这些具有⾼反应活性的中性物质很容易被吸附在较低温度的衬底表面上，发⽣非平衡反应沉积

        • 背景

          • 为解决CVD沉积温度很高的不足之处
          • 利用等离子体活化手段，提高化学反应活性，进而降低反应或沉积温度
          • 本质上属于“射频辉光放电和化学反应相结合的CVD技术”

        • 等离子体的作用

          • 将反应物中的气体分子激活成活性离子 → 降低反应所需的温度
          • 加速反应物在表面的扩散作用（表面迁移率） → 提高成膜速度
          • 对于基体及膜层表面具有溅射清洗作用，溅射掉那些结合不牢的粒子 → 加强了形成的薄膜和基板的附着力
          • 由于反应物中的原子、分子、离子和电子之间的碰撞、散射作用 → 使形成的薄膜厚度均匀

        • 主要影响因素

          • 气相成分及其分布
          • 反应区域的总压强

          • 优点

            • 更低的温度和压力
            • 提高膜厚及成分的均匀性，得到的薄膜组织致密、内应力小
            • 扩大了化学气象沉积的应用范围，提供了不同基体上制取各种薄膜的可能
            • 易于清洗反应器

          • 缺点：生产能力有限

6. 湿化学法薄膜制备技术

  • 概述

    • 特点

      • 设备简单，操作方便，成本低廉
      • 易于控制膜层的化学计量比
      • 便于制备大面积均匀薄膜

  • 溶胶-凝胶法

    • 基本概念

      • 胶体：是一种分散相粒径很小的分散体系
      • 溶胶：具有液体特征的胶体体系
      • 凝胶：具有固体特征的胶体体系

    • 定义

      • 用含高化学活性组分的化合物作前驱体，在液相下将这些原料均匀混合
      • 进行水解、缩合化学反应，在溶液中形成稳定的透明溶胶体系。
      • 溶胶经陈化胶粒间缓慢聚合，形成三维空间网络结构的凝胶，凝胶网络间充满了失去流动性的溶剂，形成凝胶。
      • 凝胶经过干燥、烧结固化制备出分子乃至纳米亚结构的材料。

    • 基本原理

      • 水解反应：M(OR)n + xH2O= M(OH)x(OR)n-x+ xROH

      • 缩聚反应

        • 失水缩聚：-M-OH + HO-M- = -M-O-M- + H2O
        • 失醇缩聚：-M-OR + HO-M- = -M-O-M- + ROH

    • 特点

      • 优点

        • 起始原料是分子级的能制备较均匀的材料
        • 较高的纯度
        • 组成成分较好控制，尤其适合制备多组分材料
        • 可降低程序中的温度
        • 具有流变特性，可用于不同用途产品的制备
        • 可以控制孔隙度
        • 容易制备各种形状

      • 缺点

        • 原料成本较高
        • 存在残留小孔洞
        • 存在残留的碳
        • 较长的反应时间
        • 有机溶剂对人体有一定的危害性

    • 工艺流程

      • 文字：易水解的金属化合物（金属醇盐或无机盐）溶解在某种有机溶剂中 → 通过水解缩聚反应形成凝胶膜 → 再通过热分解，除去凝胶中残余的有机物和水分 → 最后通过热处理形成所需要的结晶膜

Chapter3 光学光刻

0. 概述

  • 在薄膜制备技术的基础上，以光刻技术为中心的表面微纳加工技术是器件微型化与集成化的核心。

1. 光学光刻技术

  • 概述

    • 光刻技术：成功和成熟的一项技术

    • 应用

      • 集成电路芯片
      • 平板显示器

    • 在IC制造中的重要性

      • 成本比重大

      • 难易程度

        • 光刻次数
        • 所需掩模版的个数

      • 集成电路尺寸的缩小

  • 定义

    • 通过紫外光等曝光源的照射或辐射，使光刻掩膜版之下的光刻胶有选择性的曝光
    • 曝光前后的光刻胶在分子结构上产生一定的变化（高分子结构的交联或裂解），使其溶解度发生变化
    • 经过显影等过程，在光刻胶上形成微细图形（掩膜版图案转移）
    • 通过等离子刻蚀、金属蒸镀等后续工艺将所需要的微细图形转移至待加工的衬底上，最终获得所需微细图形

  • 工艺流程

    • 衬底准备

      • 要求：清洁、表面处理
      • 概述：一般要用有机溶剂脱脂，碱性清洗剂浸泡等（清洁），最后涂敷一层增粘剂
      • 目的：为保证光刻胶层与薄膜材料间有较好的粘附性，并减少针孔
      • 例子：SiO2表面的光刻

    • 涂胶

      • 要求：厚度适当、膜层均匀
      • 概述：涂胶在涂胶机上进行，胶层厚度和均匀性是涂胶的关键

    • 预烘

      • 目的

        • 增大图形的反差，增强粘附力
        • 消除胶层中的溶剂

    • 曝光（关键1）

      • 概述：曝光是光刻胶的有选择性的光化学反应过程
      • 目的：形成所需图形的“潜影”

    • 后烘

      • 目的：加速反应，使光刻胶反应完全

    • 显影（关键2）

      • 目的：将未感光的负胶或感光的正胶溶解去除，显现出所需的图形

      • 显影液

        • 正胶显影液：含水的碱性显影液
        • 负胶显影液：有机溶剂

    • 坚膜

      • 原因：显影时胶膜发生软化、膨胀

      • 概述：坚膜是对显影后的光刻件进行烘焙

      • 目的

        • 可以彻底去除显影后残留于胶膜中的溶剂
        • 增加胶层的抗蚀性和粘附性，促进热聚合反应

    • 检测

      • 概述：观察样品判断能否用于后续的微加工步骤

  • 光刻胶

    • 定义：光刻胶是一种经光照后能发生交联、分解或聚合等光化学反应的感光性有机高分子材料

    • 地位：微细加工技术的关键性基础材料

    • 分类

      • 正性光刻胶

        • 概述

          • 光化学性质是从抗溶解到可溶性
          • 正胶曝光后显影时，感光材料中分子裂解，被裂解的分子在显影液中很易溶解
          • 现有IC工艺多采用正胶

      • 负性光刻胶

        • 概述

          • 光学性能是从可溶解性到不溶解性
          • 负胶在曝光后发生交链作用形成网络结构，曝光部分在显影液中很少被溶解，而未被曝光的部分充分溶解

    • 原理

      • 主要组成

        • 主体：感光性树脂材料
        • 添加剂：增感剂、溶剂、稳定剂等附加材料

      • 结构类型

        • 线型：可溶

          • 作用力：分子间的作用力
          • 原理：长链间的结合力都相对较小，间隙较大，低分子化合物容易渗入它的内部

        • 体型：难溶

          • 作用力：通过化学键结合形成交联网络
          • 与溶剂作用：渗入体型高分子化合物会使其发生溶胀，从而减弱长键分子间的结合力

      • 感光机理

        • 概述：在外界光或热的作用下，这类高分子化合物可以产生光或热化学反应，其分子结构就可能在线型和体型之间变化，并由此相应引起其机械和物理性质的变化，如由可溶性（线型）变成不溶性（体型），或反之。

        • 分类

          • 光分解型：正胶
          • 光聚合型：负胶
          • 光交联型：负胶

      • 基本要求

        • 图形分辨率高，线条的清晰度好，尺寸精度高

          • 分辨率

            • 概述：表征两邻近的特征图形区分开来的能力，即对光刻工艺中可以达到的最小光刻图形尺寸的一种描述，是光刻精度和清晰度的标志之一

            • 描述

              • 通常以每毫米内能刻蚀出可分辨的最多线条数目来表示
              • 关键尺寸来衡量分辨率。形成的关键尺寸越小，光刻胶的分辨率越好

            • 影响因素

              • 光刻胶的种类：正性光刻胶的分辨率高于负胶
              • 分子量
              • 分布

        • 具有足够高的感光度与对比度

          • 概述：光刻胶中的感光剂是光刻胶材料中的光敏成分。即对光能发生化学反应
          • 作用：添加特定的感光剂后，可以增加感光灵敏度，而且限制反应光的光谱范围，或者把反应光限制在某一波长的光
          • 光刻胶种类的影响：负性光刻胶的感光度优于正胶

        • 与底层材料的粘附性能好：直接影响到光刻图形的精度与分辨率

        • 抗刻蚀性能好：利用光刻胶对底层材料进行有选择性的刻蚀，保护底层材料不被刻蚀

        • 对环境的稳定性好：温度、湿度、氧化气体（暗反应）时往往加入一定量的稳定剂

        • 其它，如成膜性能、致密性、针孔密度、显影后无残渣、易清除干净 等。

  • 曝光工艺

    • 地位：光刻关键工艺之一

    • 定义：利用光刻掩模对光刻胶进行有选择性的光化学反应的过程

    • 分类

      • 曝光光源分类

        • 光学曝光

          • 接触式曝光
          • 接近式曝光
          • 投影式曝光

        • X射线曝光

        • 电子束曝光

        • 离子束曝光

      • 曝光方式分类

        • 光源发出的光线
        • 光源聚集成很细的射束：电子束曝光

    • 光学曝光

      • 接触式

        • 优点：结构简单、产量高、成本低，光的衍射效应最小而分辨率高，特征尺寸小

        • 缺点

          • 容易造成掩模版和光刻胶的损伤
          • 对准困难
          • 成品率低

      • 接近式

        • 优点：以牺牲分辨率来避免接触，因此可降低工艺缺陷、延长掩模版寿命
        • 缺点：分辨率低，图形模糊，操作比较复杂
        • 应用：3μm 以上线宽

      • 投影式

        • 优点

          • 利用透镜将掩膜版图形直接投影到衬底上，避免了掩膜版与衬底接触，延长了掩膜版的寿命，对准精度高，工艺缺陷少
          • 掩膜版的尺寸可比实际尺寸大得多，克服了小图形制版的困难
          • 消除了由于掩膜版图形线宽过小而产生的光衍射效应。

        • 缺点

          • 光学系统复杂，对物镜成像要求高，设备复杂
          • 为提高分辨率，一次曝光的象场较小，采用扫描式曝光

        • 应用：3μm以下光刻

    • 曝光光源

      • 选择

        • 常规：高压汞灯
        • 其他：氙灯、镝灯、卤钨灯、准分子激光
        • 研究中：极紫外、x射线

      • 难点：光源、光学和掩膜材料

    • 先进曝光技术

      • 电子束曝光

        • 优点

          • 精度较高。电子束的斑点可以聚焦的很小，可用计算机控制，精度远比肉眼观察要高
          • 扫描电子束曝光不要掩膜版

        • 缺点

          • 设备复杂，成本较高。
          • 真空中进行，清洁度高
          • 产量小

      • X射线曝光

        • 曝光光源：X射线
        • 掩膜版：黄金或其他能挡住X射线的材料
        • 波长应用范围：0.5－2nm，分辨率高

  • 刻蚀

    • 概述：刻蚀是在光刻胶层的保护下，对其底层材料进行有选择性的去除过程

    • 地位：另一关键工艺，直接关系到光刻件的分辨率与质量

    • 分类

      • 湿法刻蚀
      • 干法刻蚀

    • 要求

      • 刻蚀速率

        • 定义：单位时间对被刻蚀材料的刻蚀厚度
        • 要求：高的刻蚀速率，可控制

      • 刻蚀选择比

        • 定义：对不同材料的刻蚀速率之比值

        • 分类

          • 被刻蚀薄膜与掩蔽膜 → 决定所需掩蔽膜的厚度
          • 被刻蚀薄膜与其下的薄膜材料 → 影响下层薄膜结构和质量

        • 要求：刻蚀比越高越好

      • 刻蚀的均匀性

        • 定义：一定面积的样品表面（包括片内、片间、批间）刻蚀速率的均匀性

        • 要求：具有高的刻蚀均匀性

        • 影响因素

          • 薄膜材料的不均匀
          • 薄膜厚度的不均匀
          • 刻蚀速率的不均匀

      • 刻蚀的方向性

        • 定义：衡量不同方向上的相对刻蚀速率

        • 分类：

          • 各向同性刻蚀：在所有方向刻蚀速率相同（化学反应）

          • 各向异性刻蚀：在不同方向刻蚀速率不同(物理刻蚀)

          • 判断

            • 图像
            • 公式：

        • 钻蚀：光刻胶层掩模之下的侧向刻蚀

        • 取向性：键密度不同

      • 衬底损伤

    • 湿法刻蚀

      • 定义：采用一定的化学溶液进行有选择性的腐蚀过程，是一个纯粹的化学反应过程

      • 优点

        • 应用范围广，适用于几乎所有材料
        • 选择比大，易于光刻胶的掩蔽和刻蚀终点的控制
        • 衬底损伤小，通常不损伤衬底
        • 操作简单，成本低，适宜于大批量加工

      • 缺点

        • 为各向同性腐蚀，容易出现钻蚀；
        • 对单晶衬底，腐蚀速率具有结晶取向；
        • 由于液体存在表面张力，不适宜于腐蚀极细的线条；
        • 化学反应时往往伴随放热与放气，导致腐蚀不均匀

      • 常见的湿法刻蚀

        • SiO2：HF、NH4F、H2O混合液

        • Si：HNO3-HF-H2O（HAC）混合液

          • HNO3：强氧化剂，将Si氧化成SiO2
          • HF：腐蚀SiO2
          • HAC：作为缓冲剂，以抑制硝酸的电离和分解
          • 蚀刻速率的调整: 可改变硝酸与氢氟酸的比例，并配合醋酸添加与水的稀释加以控制
          • 蚀刻溶液对于不同硅晶面的蚀刻速率不同 → （1 1 1）晶面化学键多，腐蚀的速率较慢

        • Si3N4：HF、H3PO4

          • 加热H3PO4
          • 采用二氧化硅做掩蔽膜

        • Al：H3PO4、HNO3、CH3COOH、H2O

        • Au：HCl、HNO3

    • 干法刻蚀

      • 定义：在辉光放电等离子体环境下，利用激发或电离的工作气体原子或分子，使其与待刻蚀材料产生化学反应、或物理溅射、或物理化学协同作用过程，去除未受光刻胶保护的底层材料

      • 类型

        • 化学性刻蚀（等离子体刻蚀）

          • 原理：等离子体产生的反应元素与被刻蚀材料发生化学反应，*需严格选择反应气体以获得高选择比

            • 等离子体：能量非常低，起刻蚀作用的是等离子体中化学活性很高的等离子体活性基团

          • 设备：等离子体刻蚀机

            • 位置：硅片放在阳极上

          • 特点

            • 优点

              • 纯化学反应进行刻蚀，选择性好
              • 对基底的损伤小

            • 缺点：高气压下，离子能量很小，各向异性差

            • 多片处理模式

          • 应用：不考虑图形转移精度的场合，如去胶工艺

          • 步骤

            • 产生等离子体
            • 等离子体活性基团与被刻蚀材料发生化学反应，生成挥发性产物
            • 反应生成物脱离被刻蚀物质表面，被真空系统抽出腔体

          • 例子：硅基材料的刻蚀

            • 刻蚀气体：CF4、C2F6、CHF3、和SF6等，其中最常用的是CF4。
            • 刻蚀选择性：CF4中掺入少量其它气体可改变刻蚀选择比

          • 工艺

            • 以化学刻蚀为主（各项同性/侧向蚀），同时也有微弱的物理溅射刻蚀作用
            • 离子的能量可以促进原子团与被刻蚀材料之间的化学反应，提高刻蚀速率，同时使刻蚀具有一定的各向异性，使分辨率有所提高
            • “侧壁保护”：刻蚀生成产物（不挥发的碳氟化物）在所有表面沉积，侧壁因为没有离子轰击，而逐渐累积碳氟化合物（聚合作用），从而降低对侧壁的刻蚀速率，进一步提高刻蚀的各向异性

          • 刻蚀气体的选择

            • SiO2：CF4
            • Si3N4：CF4
            • Si：SF6
            • 半导体材料：Cl2/Ar（主要）

        • 物理性刻蚀

          • 定义：等离子体产生的高速粒子溅射刻蚀硅片表面，一般用惰性气体。

          • 原理：利用等离子体（一般采用惰性气体）中的离子轰击，将被刻蚀材料的原子击出（能量超过原子的结合能）

          • 设备：离子铣

            • 特点

          • 特点

            • 纯粹的物理过程，电场定向，对所有材料都可实现强的各向异性刻蚀
            • 选择比差
            • 刻出物易再淀积
            • 易对下层结构造成损伤
            • 单片工艺

          • 应用：刻蚀复杂的三元和四元体系

        • 物理化学刻蚀

          • 原理：通过活性离子对衬底的物理轰击和化学反应双重作用刻蚀。兼备物理和化学作用

          • 设备：反应离子刻蚀机（RIE）

            • 实质：离子辅助刻蚀

            • 与等离子体刻蚀相比：硅片放在阴极

            • 与离子铣刻蚀相比：通入腐蚀性气体如Cl系、F系气体

            • 特点

            • 离子轰击的作用

              • 将被刻蚀材料表面的原子键破坏，加速化学反应；
              • 将再淀积于被刻蚀表面的产物或聚合物打掉，使被刻蚀表面能再与刻蚀气体接触

            • 各向异性刻蚀的实现：当使用某些混合气体，反应过程会在侧壁上形成保护性的聚合物薄膜，图形侧壁上的淀积物因未受离子轰击而保留下来，使得侧壁不被刻蚀

            • 工艺条件的影响

              • 工作气体：惰性气体，则发生纯物理的溅射刻蚀；活泼气体，则发生纯化学的等离子体刻蚀；混合刻蚀速率更大
              • 之和。
              • 气压：气压越低，物理作用越强；气压越高，化学作用越强
              • 电极：若晶片置于阴极，则物理作用较强；若晶片置于阳极，则物理作用较弱
              • 电压：电压越高，物理作用越强；电压越低，物理作用越弱

          • 特点

            • 选择比较高
            • 各向异性好，分辨率高
            • 刻蚀速率较快

          • 应用：在IC制造中应用广泛

  • 去胶

    • 定义：刻蚀结束后，光刻胶在光刻工艺中所起的图形转移的媒介作用就已完成了，因此最后需设法去除这层无用的光刻胶膜，这一工序称为去胶

    • 分类

      • 溶剂去胶

        • 定义：将刻蚀结束后的光刻件浸泡在适当的有机溶剂中，使得胶膜溶解或溶胀而脱落

        • 分类

          • 对不同类型的光刻胶，使用的去胶溶剂应有不同
          • 常有热溶剂去胶和冷溶剂去胶两种

        • 缺点

          • 效率低
          • 效果差

      • 氧化去胶

        • 定义：将刻蚀好的光刻件放入氧化去胶剂中，加热至一定温度，光刻胶膜则被氧化生成CO2和H2O,然后用水洗，即可得到清洁表面

      • 等离子体去胶

        • 定义：等离子体去胶即是在辉光放电环境中，利用氧气的等离子体去除光刻胶
        • 原理：氧气在辉光放电的激发下，形成（激发态的）氧原子，并与光刻胶产生强烈的氧化反应，使得光刻胶生成气体小分子
        • 优点：可以在刻蚀系统中完成，只是增加一路去胶气体
        • 缺点：等离子体损伤

3. 非光学光刻

  • 概述

    • 问题

      • 光学光刻的成本与代价急剧升高
      • 大量NGL面临严重障碍
      • 衍射不再对光刻分辨率起决定作用

    • 类型

      • 极紫外光刻（EUV）
      • 电子束直写光刻
      • 电子束投影式光刻
      • X射线光刻
      • 聚焦离子束光刻

  • 高能束与物质之间的相互作用

    • 非光学光刻曝光源：短波长光子（X射线）和高能电子（电子束）

    • 可能作用

      • 光电效应：光电子的能量携带几乎所有入射光子的能量（在光子能量远小于10keV时） → 低能x射线穿透能力弱
      • 康普顿效应：在这个散射过程中，入射光子的能量的一部分转移给了电子（更高能量时） → 高能x射线穿透能力强

    • x射线光刻

      • 光源：低能x射线

      • 原理：X射线光刻时，在光刻胶中产生的二次电子，其能量较电子束系统中所产生的二次电子约低一个量级 → 在X射线光刻中，能量传输的距离要小得多

        • 与电子束光刻原理的比较：X射线光刻和电子束光刻都有类似的光刻胶曝光机理

      • x射线在光刻胶中的吸收

        • X射线不是在光刻胶表面被吸收，而是穿进一定深度，直到吸收发生
        • 吸收系数是光子能量、光刻胶原子密度的函数
        • X射线对衬底材料引起的损伤也是必须考虑的问题

      • 电子与固体之间的相互作用

        • 直线通过
        • 弹性散射：主要通过与原子核之间的相互作用
        • 非弹性散射：可能产生低能二次电子

  • 电子束与X射线光刻胶

    • 原理：X射线与电子束光刻，光刻胶在电子束或X射线辐照下，会产生能量范围很宽的高密度二次电子

    • 组成：长链碳聚合物

    • 类型

      • 正性光刻胶：曝光时裂解占主导（优势）地位

        • 例：PMMA
        • 优点：很好的灵敏度，好的对比度
        • 缺点：抗等离子刻蚀能力不好，一般仅限于掩模制造或电子束直写时作为图形光刻胶用于剥离
        • 改性：PMMA的衍生物

      • 负性光刻胶：曝光时交联占主导（优势）地位

        • 优点：具有更好的灵敏度
        • 缺点：对比度较低

    • 性能指标：对比度、灵敏度、抗离子刻蚀的能力

  • 电子散射与邻近效应

    • 问题：电子束光刻只能使用薄的光刻胶层作为成像光刻胶

    • 原因：散射作用 → 邻近效应 → 线条增宽，分辨率下降

    • 电子散射

      • 定义：电子在与光刻胶分子作用时，会发生弹性与非弹性碰撞，造成电子的散射行为。散射角度与电子的能量有关，能量越高，散射角度也越大（背散射）
      • 碰撞：前散射
      • 进入衬底材料：背散射 → 逸出衬底表面

    • 邻近效应

      • 定义：由电子散射造成的曝光尺寸增大、图形边缘不清晰、侧壁陡直度下降

      • 校正

        • 几何尺寸校正：对设计好的图形，在出现邻近效应的位置
        • 曝光剂量校正：通过不同位置曝光剂量的变化，来补偿邻近效应带来的影响

  • 极紫外光刻

    • 选择原因

      • 更小的获得更高的分辨率
      • 无需分辨率增强技术
      • 对NA要求不高
      • 对存在透镜、透射、反射材料

    • EUV ：13.5nm等离子辐射光源

      • 13.5nm选择：Mo/Si 多层膜具有高的反射率.
      • 13.5nm光源：唯一可行的方式是热高密度等离子体

    • EUV光刻胶

      • 原理：胶内层电子的吸收，与分子结构无关

      • 分子吸收量=原子吸收量的加和

      • 例：PMMA

        • 特点：最高的分辨率，低感光灵敏度
        • 问题：低穿透深度
        • 解决双层光刻胶技术

    • 光刻胶LER(line edge roughness)

      • 散粒噪声：EUV光子具有高能量，由于统计光子数的涨落，更易出现高LER
      • 高感光度 → 高散粒噪声
      • 难以控制的光酸扩散
      • 光刻胶和基底中二次电子散射 → 模糊

  • 直写电子束光刻

      • 扫描模式

        • 恒流模式：初始的隧道电流设为一恒定值，针尖高度改变
        • 恒高模式：保持水平高度不变，电流发生改变

      • 应用

        • 表面原子的操控

        • STM 纳米光刻

          • 原理：利用STM针尖所产生的隧道电子对光刻胶曝光，进行无掩模的电子束直写光刻

          • 优势

            • 曝光过程更具有可控性
            • 减小了对电子束聚焦的难度与精度，曝光系统大大简化
            • 隧穿电子具有较低的能量，在光刻胶层中的散射较小，光刻分辨率更高；
            • 可以较方便使用探针阵列，实现并行曝光。

    • 原子力显微光刻（AFM）

      • 概述：利用压电单元，并结合导电悬背，可以操控探针针尖微小而精确的扫描位移

      • 应用：纳米尺度的表面成像、测量、操控原子

      • 基础

        • 原子与原子之间的相互作用力

      • 工作原理

        • 力的测量：当针尖接近样品表面时，根据胡克定律，两者之间所产生的作用力，使得悬背弯曲，据此可用于测试其作用力的大小，包括机械作用力、范德瓦尔斯作用力、毛细力、化学键合力、静电力等。
        • 高度等的测量：悬背弯曲的大小可由其表面对激光反射率通过光电二极管测试表征。在保持作用力一定的情况下，通过探针在样品表面的二维扫描，可以高精度地探测表面形貌
        • 检测器：精确检测到反射激光光斑上下左右的移动 → 此信息经反馈系统转化为控制压电扫描器的电压信号

      • 基本组成

      • 工作模式

        • 接触式

          • 概述

            • 距离：约数个Å
            • 作用力：原子间排斥力

          • 优点：排斥力对距离非常敏感，所以接触式AFM较容易得到原子级解析度

          • 缺点：探针与样品间的接触面积极小，仍会损坏样品

            • 结论：选择适当的作用力十分重要

        • 非接触式

          • 距离：约数十到数百Å

          • 作用力：长程吸引力—范德华力

            • 敏感度小 → 需要增强信噪比

          • 优点：解决接触式损害样品

          • 缺点：解析度小

        • 轻敲式

          • 原理：将非接触式加以改良，其原理是将探针与样品距离加近，然后增大振幅，使探针在振荡至波谷时接触樣品，由于样品的表面高低起伏，使得振幅改变

      • AFM 纳米光刻

        • 基于AFM扫描探针的局域化学反应过程。针尖和样品表面之间施加以脉冲电压，促使针尖附近的表面的局域反应沉积
        • 利用AFM扫描探针的机械敲击、刮擦等作用，实现纳米尺度图案的修改或形成
        • 利用AFM扫描探针作为蘸笔，蘸取其它溶液进行书写纳米尺度图案

  • 干涉光刻

    • 原理：两个及以上的相干光波构成干涉图样。在曝光过程中，其对应的强度分布可使光刻胶选择性曝光，显影后形成记录在光刻胶中的图样 → 通过叠加不同光束组合，不同的结构便得以制备出来

  • 自组装技术

    • 定义：分子及纳米颗粒等结构单元在平衡条件下，通过非共价键作用自发地缔结成热力学上稳定的、结构上确定的、性能上特殊的聚集体的过程

    • 特点：将不需要外力的作用自动进行到某个预期的终点

    • 分类

      • 纳米结构自组装体系：通过弱的和较小方向性的非共价键，如氢键、范德华力和弱的离子键协同作用把颗粒、原子、离子或分子连接在一起构筑成一个纳米结构或纳米结构的花样
      • 分子自组装体系：装指分子与分子在平衡条件下，依赖分子间非共价键力自发地结合成稳定的分子聚集体的过程

    • 例：胶体晶体

  • 纳米压印技术

    • 定义：通过机械作用力，利用有机高分子材料（压印胶）形变或流变位移，将模板表面的纳米图案转移至胶层上的图形转移技术

    • 核心技术：工艺 + 材料 + 模具 + 设备

    • 分类

      • 软压印技术

        • 定义：利用高分子弹性模板或印章，在衬底表面复制微纳结构的技术

        • 微接触印刷；通过微接触方式，将模板微结构表面的高分子等溶液，印刷转移到衬底表面上

        • “墨水”的选择

          • 功能高分子
          • 纳米颗粒
          • 微米颗粒

        • 主要特点

          • 优点

            • 方便、便宜，分辨率较高，适于非平面
            • 易于为化学、生物、材料领域所接受

          • 缺点：对细小尺度图形的对准、图形结构的变形、同IC工艺的兼容性 等问题尚待解决

      • 热塑型纳米压印

        • 定义：加热冷却固化型

      • 紫外曝光纳米压：压印过程中，纳米结构的定型，即压印胶的固化，是采用紫外曝光的方式。与前述加热冷却固化型相比，压印胶种类不同；胶层可采用旋涂，也可以滴涂
